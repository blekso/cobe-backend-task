# Node, Typescript Backend task for Internship at COBE

### Dependencies

* Mongoose, nodemon
* Windows 10

### Description
Before running the project user has to have mongo daemon launched in the background!

### Installing

```
npm i
```

### Run the program

Run dev 
```
npm run dev
```

Build 
```
npm run build
```

Run build
```
npm run start
```
